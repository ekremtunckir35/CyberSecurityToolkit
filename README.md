## 🚀 Araçları Kullanmak

```bash
git clone https://github.com/ekremtunckir35/CyberSecurityToolkit.git
cd CyberSecurityToolkit
pip install -r tools/requirements.txt

# Port tarayıcı
python tools/port_scanner.py --target 192.168.1.1 --ports 1-1024

# Banner grabber
python tools/banner_grabber.py --target 192.168.1.1 --port 80
```
